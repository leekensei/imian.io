### 6. 코드

인라인 코드: `console.log('Hello World!')`

코드 블록:

```javascript
function greeting(name) {
  return `Hello, ${name}!`;
}

// 함수 호출
console.log(greeting("Developer"));
```

라어라어

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 피보나치 수열의 첫 10개 숫자 출력
for i in range(10):
    print(fibonacci(i))
```
